# Bio Profile App

A React-based genetic intelligence dashboard with AI companion features.

## Features

- Interactive biological profile dashboard
- AI-powered genetic insights
- Chatbot companion for wellness guidance
- Daily tasks and rewards system
- Privacy and security controls
- Personalized recommendations

## Tech Stack

- React 18
- Tailwind CSS
- Create React App

## Development

```bash
npm install
npm start
```

## Deployment

This app is configured for deployment on Vercel:

```bash
npm run build
```
